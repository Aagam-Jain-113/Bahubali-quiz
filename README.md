## Usage
To run locally

1. Clone the repository
```
$ git clone https://github.com/Aagam-Jain-113/Bahubali-quiz.git
```
2. Install packages
```
$ npm install --save readline-sync
$ npm install --save chalk
```
3. Run the program
```
node index.js
```
Now you are Good to Go!
